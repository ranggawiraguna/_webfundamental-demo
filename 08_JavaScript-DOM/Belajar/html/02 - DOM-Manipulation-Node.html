DOM MANIPULATION NODE

document.createElement()

(Method ini berfungsi untuk membuat suatu elemen HTML baru)

(elemen yang dibuat dengan ini tidak langsung masuk kedalam document, perlu ada beberapa langkah supaya dapat memunculkan kedalam tampilan)

document.createTextNode()

(Method ini berfungsi untuk membuat suatu teks yang akan dimasukkan kedalam innerHTML dalam suatu elemen)

(method ini biasa diapakai setelah membuat elemen dengan createElement())

node.appendChild()

(Method ini berfungsi untuk menambahkan suatu elemen kedalam suatu parent)

(Method sebelumnya hanya membuat object saja, nah selanjutnya method ini bekerja untuk memasukkannya kedalam sebuah node)
