(Pencampuran warna yang terjadi secara linear atau lurus)

(Value ini dapat merubah arah gradiasi warna yang terjadi menggunakan derajat lingkaran)

90deg
135deg

Linear Gradient :: Direction

(Value ini dapat merubah arah gradiasi warna yang terjad menggunakan nama arah tujuannyai)

to left
to top left
to right

Linear Gradient :: Color Stop

(Value ini dapat menghentikan bagian pada warna tertentu)

Linear Gradient :: Entirely

(Penggunaan keselurahan pada Linear Gradient)


Radial Gradient :: Normal

(Pencampuran warna yang terjadi secara radial atau melingkar)

Radial Gradient :: Color Stop

(Value ini dapat menghentikan bagian pada warna tertentu)

Radial Gradient :: Type

(Value ini dapat menentukan bentuk dari radial pada warna gradient)

ellipse
circle

Radial Gradient :: Range

(Value ini dapat menentukan titik berhentinya suatu gradient)

closest-side/sisi-terdekat
farthest-side/sudut-terjauh

Radial Gradient :: Position

(Value ini dapat merubah pusat dari gradient yang tebentuk)

at top left
at bottom right
