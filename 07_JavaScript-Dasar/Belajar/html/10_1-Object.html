OBJECT

Basic Object

(Objek adalah sebuah variable yang memiliki atribut, dan setiap atributnya berisi sebuah nilai)

(bahkan kita dapat membuat objek yang didalamnya berisi sebuah objek juga)

Object Literal

(Object yang dideklarasikan secara langsung dan disimpan pada satu variable)

(Pembuatan object seperti ini sangat tidak eektif, karena jika ingin membuat object lain dengan property yang sama, maka kita harus menuliskan kode program pembuatan object nya berulang kali)


Function Declaration

(Sama seperti pembuatan object literal, hanya saja proses pembuatan objek dimasukkan kedalam function yang mengembalikan nilai objek baru)

(objek baru yang dikembalikan akan tersimpan pada suatu variable, sehingga kita tidak perlu menulis ulang kode proses pembatan object jika ingin membuat objek baru)


Constructor Function

(Cara ini adalah cara yang paling efektif, karena function akan membuat attribut pada variable this)

(this disini dimaksudkan pada variable yang memanggil funtion menggunakan keyword new)

(this juga terdapat pada variable global, yaitu window itu sendiri)
