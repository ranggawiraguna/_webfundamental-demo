FUNCTION ARRAY

Join

(Method ini dapat menggabungkan semua isi array menjadi satu string)

(Defaultnya isi array akan digabung dan dipisahkan dengan koma, tetapi kita dapat merubahnya dengan mengirim parameter pada saat memanggil join)

Push & Pop

(Method ini dapat memanupulasi isi array pada index terakhir)

(Push digunakan untuk menambahkan, sedangkan Pop digunakan untuk menghapus)

Unshift & Shift

(Method ini dapat memanupulasi isi array pada index pertama)

(Unshift digunakan untuk menambahkan, sedangkan Shift digunakan untuk menghapus)

Splice

(Method ini dapat memanipulasi isi array pada bagian index tertentu)

(dapat digunakan untuk menghapus elemen, maupun menambahkan elemen pada indeks tertentu)

Slice

(Method ini dapat memotong/mengambil beberapa bagian index yang terurut pada array)

(Method ini tidak merubah isi pada array tetapi meng-copy nya sesuai index slice yang ditentukan)

ForEach

(Method ini dapat melakukan perulangan ke semua elemen yang ada pada array)

(Method ini hanya menelusuri semua elemen array, tetapi tidak mengembalikan apapun setelahnya)

Map

(Method ini dapat melakukan perulangan ke semua elemen yang ada pada array)

(Method ini akan mengembalikan array baru hasil dari perulangan pada array yang di looping)

(Jika pada looping setiap elemen array dirubah maka map akan mengembalikan nilai perbahan nya tersebut dalam bentuk array juga)

Sort

Filter

(Method ini dapat mencari beberapa elemen yang sesuai dengan ketentuan pada filter)

(Method ini akan mengembalikan sebuah array berisi elemen yang sesuai dengan ketentuan)

Find

(Method ini dapat mencari satu elemen yang sesuai dengan ketentuan pada filter)

(Method ini akan mengembalikan satu nilai yang sesuai dengan ketentuan)

FindIndex

(Sama halnya dengan find, akan tetapi method ini mengembalikan index dari elem yang didapat bukan elemen itu sendiri)
