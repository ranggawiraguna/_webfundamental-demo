Declaration & Expression

Function Declaration

(Function yang dideklarasikan secara langsung dan dapat dipanggil sebelum atau sesudah function ini dideklarasikan)

(Berdasarkan cara pemanggilannya function ini bisa dibilang lebih fleksibel)


Function Expression

(Function yang dideklarasikan tanpa nama tetapi disimpan pada sebuah variable)

(Function ini hanya dapat dipanggil setelah pendeklarasian variable yang menyimpan dfunction tersebut)

(Berdasarkan kinerjanya function ini lebih powerfull, misalnya dapat dijadikan sebuah argument pada function lain dengan mengirim variable yang mengirim functionnya)
