VAR , LET , CONST

VAR

(tipe variable dengan var akan membuat variable menerapkan proses hoisting yang ada pada javascript)

(tipe variable ini tidak efektif karena variable var akan dideklarasikan telebih dahulu dengan nilai undefined)

(kita dapat langsung mengakses variable var walaupun dideklarasikan di setelah statement pemanggilannya)

LET

(tipe variable ini tidak menerapkan proses hoisting, karena variable let harus dideklarasikan terlebih dahulu sebelum dipanggil, jika tidak maka akan menghasilnya statement error)

(tipe variable ini lebih efektif, dan juga proses pembuatannya seperti bahasa pemrograman lainnya sehingga tidak mengbingungkan programmer)

CONST

(tipe ini sama seperti let yang tidak menerapkan proses hoisting, hanya saja value dari tipe variable ini tidak dapat dirubah (fix))

(value dari variable ini tidak dapat dirubah kecuali, jika variable const bertipe array atau object, kita masih dapat mengubah nilai index dari array atau nilai attribut dari object)
