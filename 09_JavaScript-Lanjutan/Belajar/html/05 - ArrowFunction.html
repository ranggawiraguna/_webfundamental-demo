Basic Arrow Function

(Arrow Function adalah versi yang lebih ringkas dari pembuatan function)

(berbeda dengan dunction declaration atau expression, arrow function tidak di inialisasi dari awal seperti halnya function lain)

(jadi block scope pada arrow function terdapat pada saat pendeklarasian function tersebut)

This Pada Arrow Function

(Pada arrow function tidak menerapkan keyworad this, jadi arrow function tidak mengenali this)

(Karena arrow function tidak dijalankan oleh hoisting, maka block scope dari arrow function berbeda)

(Jika ternayata didalam arrow function memanggil this, maka function akan mencari ke scope 1-tingkat diatasnya dan seterusnya, hal ini yang membuat arrow function berbeda dari function lain)
