TEMPLATE LITERAL

Basic Template Literal

(Sebuah penulisan string menggunakan tanda backtick(`))

(String ini memungkinkan untuk menulis string dengan banyak baris, memasukkan isi variable dan sebagainya)

Multi-line String

(Dengan menuliskan string menggunakan template literal, kita dapat membuat string yang lebih dari satu baris tanpa harus menggunakan c-escape enter atau dipecah dengan concat string)

(Hal ini memudahkan kita dalam menuliskan baris string yang lebih dari satu dengan satu template backtick)


Embedded Expression | Expression Interpolation

(Dengan template literal kita juga dapat menuliskan suatu expression didalam template literal)

( cara penulisan expression pada template literal adalah seperti ini ${expression} )

( Expression dapat berupa variable,function return,dll)

HTML Fragments

(Dengan template literal kita juga dapat membuat script html didalamnya)

(Dengan ini juga dapat langsung memasukkan data yang bersangkutan kedalam html scriptnya secara langsung)
