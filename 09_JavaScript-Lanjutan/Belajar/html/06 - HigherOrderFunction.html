HIGHER ORDER FUNCTION

Definisi

(Higher Order Function adalah function yang menerima argument sebuah function lain, atau bisa function yang mengembalikan nilai sebuah function lain(closure))

(Sedangkan function yang menjadi argument atau return value nya disebut sebagai callback)

Filter

(Method ini dapat menyeleksi isi array yang sesuai dengan ketentuan pada return)

(Jika menggunakan looping untuk meyeleksi array tentunya akan memerlukan baris kode yang lumayan banyak)

(Method ini juga termasuk Higher Order Function karena menerima sebuah callback untuk menseleksi arraynya)

Map

(Method ini dapat memetakan isi array dan melakukan perubahan pada setiap elemen, setalh itu method ini akn mengembalikan array baru hasil dari prosesnya)

(Jika menggunakan looping untuk merubah isi dan menduplikat array tentunya akan memerlukan baris kode yang lumayan banyak)

(Method ini juga termasuk Higher Order Function karena menerima sebuah callback untuk memproses setiap elemen arraynya)

Reduce

(Method ini dapat melakukan suatu operasi untuk semua elemen array, jadi setia elemen dioperasikan secara terurut sampai akhir (misal pejumlahan total isi array))

(Jika menggunakan looping untuk mengoperasikan semua array tentunya akan memerlukan baris kode yang lumayan banyak)

(Method ini juga termasuk Higher Order Function karena menerima sebuah callback untuk memproses setiap elemen arraynya)
